TOPL

TOPL is a language for specifying temporal properties of Java programs. Its main design goal is simplicity. Users know how to use it immediately after seeing a couple of examples.

property NullInCollection
  observe <java.util.Collection.add>
  prefix <java.util.Collection>
  start -> start: *
  start -> error: *.add(<null>)

The theoretical underpinnings are based on finite-memory automata.

The command toplc -i javaProjectDirectory properties.topl instruments a Java project. The resulting bytecode reports to stderr if a property reaches the error state.
